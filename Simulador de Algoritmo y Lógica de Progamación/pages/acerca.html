Acerca del Proyecto

Este simulador de ejercicios fue desarrollado para facilitar la práctica interactiva de preguntas y respuestas sobre la materia de Algoritmo y Lógica de Programación. Está diseñado para ser fácil de usar y ofrece retroalimentación inmediata sobre el progreso del usuario.
